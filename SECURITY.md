# Security Policy

## Supported versions

`swift-common` is pre-1.0. Only the `main` branch is supported — fixes land
there and consumers pin to the git URL.

| Version | Supported |
| ------- | --------- |
| `main`  | ✅        |
| tagged pre-1.0 releases | ⚠️ best-effort |

## Reporting a vulnerability

Please report security issues **privately** — do not open a public issue.

- Preferred: [GitHub private vulnerability reporting](https://github.com/rtorcato/swift-common/security/advisories/new)
- Email: rtorcato@me.com

Include repro steps, affected platform(s), and the Swift/Xcode version where
you observed the issue.

## Response window

- Acknowledgement: within **7 days**.
- Assessment and fix plan: within **30 days** of acknowledgement.
- Disclosure: coordinated — the advisory is published once a fix is available
  on `main`, crediting the reporter unless they prefer otherwise.
